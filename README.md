# Gene-Pesticide Exposure Interactions in Parkinson's Disease repository

**Author**: Samuel Jorquera


In this repository you will find the codes and scripts used for the generation of the manuscript "Gene-Pesticide Exposure Interactions in Parkinson's Disease: A Systematic Review and Population-Specific Allele Frequency Analysis".

For population-level analysis of GxPE loci frequencies scripts and data cand be found in the [GxPE_Freq](/GxPE_Freq) directory.
